# deepseek-cookbook
DeepSeek 学习代码

## 一 作者环境说明
- Python版本：3.11.9
- 操作系统：Windows 11/10

## 二 依赖清单
### 1.1 requirements.txt
项目根路径下的 requirements.txt 文件详细记录了项目的依赖包版本信息。
### 1.2 pip 命令清单
也提供了 pip 命令清单，用于安装依赖包。详见项目根路径下的 pip_command.txt

## 三 Jupyter 使用
使用 Jupyter Lab 方便调试，节省 API 调用费用

启动 Jupyter Lab
```commandline
jupyter lab
```

## 四 Streamlit 使用
运行 streamlit 应用命令
```commandline
streamlit run streamlit_app.py
```
